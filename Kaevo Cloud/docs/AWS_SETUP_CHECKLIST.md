# AWS Setup Checklist

Do these before deploying anything.

## Account safety

- Turn on MFA for the AWS root account.
- Do not use root for daily work.
- Create a separate admin/developer user or role.
- Use least privilege later when the project is stable.

## Cost safety

- $5

- $25
