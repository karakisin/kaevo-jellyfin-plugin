## Security-change relationship

Security-related genuine regressions were found in provider-identity persistence, idempotency, endpoint verification status, missing-item write gating, Seerr permission masks, deferred identity review state, and recommendation influence calculation. The remaining failures were intentional product-contract drift, one broken fixture, or one simulator-state precondition. No nondeterministic failure was identified.

## Release gate

The required unit-test release gate is a complete `iOS Kaevo v2Tests` run on the iPhone 17 simulator with code signing disabled. Physical-iPhone playback and PiP evidence is tracked separately and is not inferred from this unit-test result.
