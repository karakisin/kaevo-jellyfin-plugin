# Kaevo Travel Downloads — UX Copy and Flows

## Primary CTA

Preferred label:

```text
Download for Travel
```

Acceptable alternate:

```text
Download Offline
```

## Quality picker

### Header

```text

```



```text

```

## Quality cards

### 480p

```text
Storage Saver 480p
Good enough for phone screens. Downloads faster and lets you fit more.
```

Optional detail line:

```text
Best if you want more episodes or need the download to finish sooner.
```

### 720p

```text
Recommended 720p
Sharper and better for most phones. Uses more storage than 480p.
```

Optional detail line:

```text
Best balance of quality, storage, and download time.
```

### 1080p

```text
High Quality 1080p
Great for iPad, hotel TVs, or larger screens. Smaller than 4K but still sharp.
```

### Original

```text
Original Quality
Best available quality. Largest file and may take much longer.
```

## Estimate labels

Use soft wording:

```text
Estimated size
Estimated download time
Estimated total time
```

Avoid exact promises like:

```text
Will finish in 8 minutes
```
